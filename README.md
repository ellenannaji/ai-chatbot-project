# 🤖 AI Chatbot – Full Stack Project

A modern AI chatbot web application built using **Flask, OpenAI API, and Vanilla Frontend (HTML, CSS, JavaScript)**.  
It supports **real-time streaming responses like ChatGPT**, voice input, dark/light mode, and chat history saving.

---

## ✨ Features

- 💬 Real-time AI responses (streaming like ChatGPT)
- 🎤 Voice input support (Speech Recognition)
- 🌗 Dark / Light theme toggle
- 💾 Chat history saved in browser (localStorage)
- ✨ Typing cursor animation
- 🎨 Glassmorphism modern UI design
- 📱 Fully responsive design
- ⚡ Fast Flask backend API

---

## 🛠️ Tech Stack

### Frontend

- OpenAI GPT-4o-mini API
